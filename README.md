# Kivymd-Viewer
Simple kivymd and kivy ( auto-kivy-language-viewer )
### support :
  - ubuntu
  - kali

### install :
```bash
apt-get install python3-pip
pip3 install --upgrade pip
sudo python3 setup.py
```
