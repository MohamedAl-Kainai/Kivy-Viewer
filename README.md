# Kivy-Viewer
Simple kivy and kivymd ( auto-kivy-language-viewer )
### support :
  - ubuntu
  - kali

### install :
```bash
apt-get install python3-pip
pip3 install --upgrade pip
git clone https://github.com/No-Name-404/Kivy-Viewer
sudo python3 setup.py
```
